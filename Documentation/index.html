Stylesheets

Here's a list of the stylesheet files I'm using with this template, you can find more information opening each file:

File Name Description
reset.css The first one is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this.
typography.css This file also contains some Heading styles, such as h1,h2.h3, Home title, Tag, General Content.
responsive.css uses CSS3 media queries, an extension of the @media rule, Media queries allow the page to use different CSS style rules based on characteristics of the device the site is being displayed on, most commonly the width of the browser.
style.css This file contains most of the css used by the theme .
prettyPhoto.css which contains some code used by prettyPhoto.
css/nivo.slider.css which contains Nivo Slider CSS // For Abouts Me Photo Sliding
css/font-awesome.css font-awesome-ie7.css which contains font-awesome CSS // For icon
